## Full Description

Free YouTube transcript extractor that runs entirely in your browser. No account, no credits, no server, no limits.

Everything youtube-transcript.io charges $9.99/month for, this does for free:

- Single video, playlist, or channel transcript extraction
- Export transcripts in six formats including subtitles and structured data
- AI-powered summaries, quizzes, flashcards, mind maps, study guides (bring your own API key or use Chrome built-in AI)
- Speaker detection, chapter extraction, filler word removal
- Transcript search, highlights, and notes
- Works offline once fetched

Your data stays in your browser. No tracking. No analytics. Network tab shows zero requests to anything except YouTube and your own AI provider.

Built by a student. Open source. PRs and suggestions welcome.

## Privacy
This extension does not collect, store, or transmit any user data. API keys are stored locally in chrome.storage. AI requests go directly from your browser to the provider you configure. There is no backend server.
